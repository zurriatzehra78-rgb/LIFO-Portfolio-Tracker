# LIFO-Portfolio-Tracker

This project demonstrates the use of Stack (LIFO) data structure using Streamlit.

## Features
- Buy and Sell stock transactions
- Stack operations (Push, Pop, Peek)
- Live stock prices using yfinance
- Transaction history tracking

## Technologies Used
- Python
- Streamlit
- Pandas
- yfinance

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
